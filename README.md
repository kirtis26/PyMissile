# PyMissile

Баллистический расчет ЗУР "бескрылой" схемы с однорежимной ДУ методом наведения с расчетом аэродинамики ракеты на траектории ее полета

**Исходная мат. модель: https://github.com/TovarnovM/neuro_missile**
